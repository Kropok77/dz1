# Описание проекта
Проект написал студент 1 курса СКБ 251 Ефимюк Ю. Д.
Проект представляет собою реализацию программы книжной библотеки. В проекте применены основы ООП.
## Требования 
C++17+, CMake3.10+
## Инструкия по запуску
1. Перейти в папку library.proj
1. выполнить команду `cmake`
1. запустить файл `library`
